# 🎨 Skiller Community Skills

Cursor AI Agent Skills 社区共享仓库 - 分享和发现 Agent 技能

## 📁 仓库结构

```
skills/
├── <category>/
│   └── <skill-name>/
│       └── SKILL.md
└── index.json          # 自动维护的技能索引
```

## 🚀 如何贡献

1. 在 Skiller Dashboard 中配置此仓库和你的 GitHub Token
2. 选择你的本地 Skill，点击「上传到社区」
3. 你的 Skill 将出现在社区广场中供他人使用
